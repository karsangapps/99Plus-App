Q.3 Consider the following Assertion and Reason:

Assertion (A): The function f(x) = |x − 1| is not differentiable at x = 1.

Reason (R): The left-hand derivative and right-hand derivative of f(x) at x = 1 are equal.

Select the correct option:

Conceptual Bridge
Surgical Fix

ONE-LINE LOGIC FIX

For |x - 1|, the LHD = -1 and RHD = +1. Since LHD ≠ RHD, the Reason is false — making Option C correct.

PATTERN TO REMEMBER

Modulus functions always create a “kink” — always check LHD vs RHD at the corner point before assuming differentiability.
